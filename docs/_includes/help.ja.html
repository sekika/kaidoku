• マスを選んで数字を入れてください。
  • 同じマスに2つ以上の数字を入れると、メモになります。
  • 同じ数字を入れるとその数字が消えます。
  • X ですべての数字が消えます。
  • H で解き方のヒントを表示します。ナンプレの解法も参照してください。
  • >>> 続きを読む
  • R で最初からやり直します。
  • B で直前の手を取り消します。ミスの修正と数字の仮定を戻すときに使えます。
  • P で鉛筆モードになります。鉛筆モードでは、数字の重複自動チェックやヒント機能などは使えません。紙と鉛筆で解く状態と同じとなります。
  • 新しい問題は、レベルを選択して問題番号を入力することで選ぶことができます。
  • 問題番号0を選択すると、ランダムな問題が表示されます。数字と回転のシャッフルがされています。
  • ランダムな問題(問題番号0)を鉛筆モードで開始すると、タイムトライアルが開始します。
  • ブラウザを閉じても立ち上げ直せば続きができます。現局面はブラウザに保存されています。
  • PCからキーボードを使うこともできます。キーボードでは c で現局面をクリップボードにコピーして解独で(解析できます。
